Modelos C4

Modelo C4 nível 1:

Mostra o sistema de software que está sendo construindo e como ele se encaixa no mundo em termos das pessoas que o utilizam e dos outros sistemas de software com os quais ele interage.

Modelo C4 nível 2:

Um diagrama de container que amplia o sistema de software e mostra os containers (aplicativos, armazenamentos de dados, microservices, etc.) que compõem esse sistema de software.

Modelo C4 nível 3:

Um diagrama de componentes, amplia um container individual para mostrar os componentes dentro dele.
